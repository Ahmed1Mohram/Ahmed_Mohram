# Audio Files

This directory contains audio files for the education platform.

## File: mansah.mp3

This is a voice recording saying "منصة" (Platform) that plays when the "أنا مين؟" (Who am I?) button is clicked.

## Instructions

1. Place your audio file named `mansah.mp3` in this directory.
2. Make sure it's a short, clear recording of someone saying "منصة".
3. The file should be in MP3 format with reasonable quality (128kbps or higher).
4. Keep the file size small (under 1MB) for fast loading.
